Sous-tables

Nous créons ici deux sous-tables, une pour "automatic", l’autre pour "manual", avec le code suivant :

mtcars_auto = subset(mtcars, subset = trans == "automatic")
mtcars_manu = subset(mtcars, subset = trans == "manual")
